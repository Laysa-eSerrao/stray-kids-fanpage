🎵 Stray Kids — Guia para Novos STAYs

Página web interativa para quem quer conhecer o grupo de K-pop Stray Kids. Feita com HTML, CSS e JavaScript puro.

📋 Conteúdo da página

Membros — cards interativos com perfil completo de cada um dos 8 membros (clique para expandir)
Sobre — história do grupo, estilo musical e conquistas recentes
Discografia — álbuns de 2018 a 2024, com destaque para os #1 no Billboard
Unidades — 3RACHA, Dance Racha e Vocal Racha explicadas

🚀 Como rodar localmente
bash# Clone o repositório
git clone https://github.com/Laysa-eSerrao/stray-kids-fanpage.git

# Abra o arquivo no navegador
open index.html
Ou simplesmente arraste o index.html para o seu navegador.

🌍 Como publicar com GitHub Pages

Acesse Settings do repositório
Vá em Pages (menu lateral)
Em "Branch", selecione main e clique em Save
Aguarde alguns minutos — sua página estará em https://SEU-USUARIO.github.io/stray-kids-fanpage/

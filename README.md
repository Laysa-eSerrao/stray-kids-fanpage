# 🎵 Stray Kids — Guia para Novos STAYs

> Fan page interativa e completa sobre o Stray Kids, feita com HTML, CSS e JavaScript puro — sem frameworks, sem dependências.

---

## 🌐 Ver a página

👉 **[laysa-eserrao.github.io/stray-kids-fanpage](https://laysa-eserrao.github.io/stray-kids-fanpage/)**

---

## 📋 Abas e Conteúdo

| Aba | Conteúdo |
|---|---|
| **Membros** | Cards com foto do SKZoo, perfil completo e detalhe expandível de cada membro |
| **Galeria** | Fotos reais dos 8 membros com lightbox |
| **Curiosidades** | 12 cards com fatos interessantes sobre o grupo |
| **Sobre** | História, estilo musical, conquistas e SKZoo com personagens oficiais |
| **Discografia** | Álbuns de 2018 a 2025 com capa, faixa título e número de faixas |
| **Unidades** | 3RACHA, Dance Racha e Vocal Racha |
| **Stream e Votação** | Links para Spotify, YouTube, MelOn, Genie, Bugs + MAMA, AMAs, SMA, AAA |
| **Tours** | Histórico completo de tours desde 2022 + novidades 2026 |
| **Conteúdo** | Séries oficiais (2 Kids Room, RACHA LOG, SKZ CODE...) e filme de concerto |
| **MVs** | 9 MVs com thumbnail real do YouTube e link direto |
| **História** | Linha do tempo interativa de 2017 a 2026 com animações |
| **2026** | Novidades: single de aniversário, fan meeting, Rock in Rio e mais |

---

## ✨ Funcionalidades

- 🌙 **Modo claro/escuro** com preferência salva no navegador
- ⬆️ **Botão voltar ao topo** com animação suave
- 🎯 **Contador regressivo** para o Rock in Rio 2026
- 📤 **Share buttons** para Twitter/X, WhatsApp e copiar link
- 🎬 **Lightbox** para fotos da galeria
- 🎞️ **Linha do tempo** com animação de entrada ao rolar
- 📱 **Layout responsivo** para mobile
- 🔍 **SEO otimizado** com Open Graph para WhatsApp e Twitter
- 💀 **Página 404 personalizada** com tema SKZ
- 📊 **Google Analytics** integrado
